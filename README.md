# EmojiArt

The aim of this app is to make emoji drawing for all Apple's platform from iOS to iPadOS to macOS. Everything is iCloud synced. My main purpose was to learn hot to drag and drop files in thsi project. The project is built using [Swift](https://swift.org/documentation/#the-swift-programming-language) and uses SwiftUI.
